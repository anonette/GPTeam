 def get_format_instructions(self) -> str:
        tool_names = ", ".join([tool.name for tool in self.tools])
        return f"Your response should use the following format:\n\nTask: the task you must complete with explanation how it serves your main agenda\nThought: your thoughts about what you should do to achieve this\nAction: the action to take, which must be one of these words: [{tool_names}]\nAction Input: the input to the action\nObservation: the result of the action\n... (this Thought/Action/Action Input/Observation should not repeat)\nThought: 'I realized new thing.'\nFinal Response: the final response to the task\n\nIf you are not ready with a final response, then you must take an action.\n\nIf you determine that you are incapable of completing the task with the tools you have been given, return 'Final Response: I need to ask the rest', however, if you are in dialog with another character, responses such as 'I don't know' are valid responses.\nIf the task is complete and no specific response is needed, return 'Final Response: Done'"
